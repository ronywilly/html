Etiquetas básicas

  • <!DOCTYPE>: versión de HTML
  • <html>: define el inicio y final de la página
  • <head>: encabezado de página
  • <title>: título de la página
  • <body>: cuerpo de la página
  • <footer>: pie de página

Ejemplo:

<!DOCTYPE>

<html>

<head>

<title> Título de la página web</title>

</head>

<body>

<header>Encabezado</header>

<nav>Menú navegación</nav>

<main>Contenido principal</main>

<p>Párrafos</p>

<section>Secciones</section>

<article>Artículos</article>

<aside>Publicidad</aside>

</body>

<footer>

Contenido del pie de página

</footer>

</html>

Etiquetas semáticas

  • <header>: cabecera de una página o sección
  • <h1>: título principal
  • <h2-h6>: subtítulos en jerarquía
  • <nav>: menús principales, índices
  • <main>: contenido principal
  • <section>: agrupa contenido
  • <article>: contenido independiente
  • <aside>: contenido relacionado indirectamente

Ejemplo:

<header>Encabezado</header>

<h1>Título de la página web</h1>

<h2-h6>Subtítulos de la página web</h2-h6>

<nav>Menú navegación</nav>

<main>Contenido principal</main>

<p>Párrafos</p>

<section>Secciones</section>

<article>Artículos</article>

<aside>Publicidad</aside>

Etiquetas para texto

  • <strong>: texto en negrilla
  • <b>: texto en negrilla
  • <em>: texto en cursiva
  • <i>: texto en cursiva
  • <del>: texto tachado
  • <s>: texto tachado
  • <ins>: texto subrayado
  • <u>: texto subrayado
  • <q>: texto entre comillas

Ejemplo:

<strong>Uso de negrilla con la etiqueta <strong> o la etiqueta <b></strong>

<em>Uso de cursiva con la etiqueta <em> o la etiqueta <i></em>

<del>Uso tachado con la etiqueta <del> o la etiqueta <s></del>

<ins>Uso de subrayado con la etiqueta <ins> o la etiqueta <u></ins>

<q>Uso de comillas con la etiqueta <q></q>

Etiquetas para multimedia

  • <img>: define una imagen en la página
  • <audio>: define un audio en la página
  • <video>: define un video en la página
  • <source>: define el origen de archivos

Ejemplo:

<img scr=imagen.png width=50 height=50>

<audio controls>

<source src="audio.mp3" type="audio/mpeg">

</audio>

<video controls width="400">

<source src="video.mp4" type="video/mp4">

</video>

Etiquetas variadas

  • <p>: define un párrafo de texto
  • <a>: define un enlace interno o externo
  • <br>: salto de línea
  • <ol>: listas ordenadas
  • <ul>: listas no ordenadas
  • <li>: línea de una lista
  • <table>: define una tabla
  • <tr>: número de filas de una tabla
  • <td>: número de columnas de una tabla
  • <th>: fila como encabezad en negrilla
  • <cite>: define citas de texto
  • <abbr>: textos en abreviatura
  • <blockquote>: agregar sangría a un párrafo
  • <textarea>: ingreso de texto
  • <div>: dividir y agrupar contenido

<p>Este es un párrafo...</p>

</p>

<ol>

<li>Línea 2</li>

<li>Línea 3</li>

</ol>

<ul>

<li>Línea 1</li>

<li>Línea 2</li>

<li>Línea 3</li>

</ul>

<table>

<tr>

Etiquetas para formularios

  • <form>: define un formulario
  • <label>: texto para identificar los input
  • <select>: menú de opciones
  • <option>: opciones de un menú
  • <input>: ingreso de datos al formulario
    • <input-text>: dato tipo texto
    • <input-email>: dato tipo correo electrónico
    • <input-checkbox>: dato o datos seleccionados
    • <input-radio>: dato seleccionado
    • <input-date>: dato tipo fecha
    • <input-password>: dato tipo contraseña

Ejemplo:

<form action="/procesar" method="POST">

<label for="nombre">Nombre:</label>

<input type="text" id="nombre" name="nombre"><br>

<label for="email">Correo electrónico:</label>

<input type="email" id="email" name="email"><br>

<label>Intereses:</label><br>

<input type="checkbox" id="deportes" name="intereses" value="deportes">

<label for="deportes">Deportes</label><br>

<input type="checkbox" id="musica" name="intereses" value="musica">

<label for="musica">Música</label><br>

<input type="checkbox" id="tecnologia" name="intereses" value="tecnologia">

<label for="tecnologia">Tecnologia</label><br>

<label>Género:</label><br>

<input type="radio" id="masculino" name="genero" value="masculino">

<label for="masculino">Masculino</label><br>

<input type="radio" id="femenino" name="genero" value="femenino">

<label for="femenino">Femenino</label><br>

<label for="fecha">Fecha:</label>

<input type="date" id="fecha" name="fecha"><br>

<label for="password">Contraseña:</label>

<input type="password" id="password" name="password"><br>

<label for="pais">País:</label><br>

<select id="pais" name="pais">

<option value="mx">México</option>

<option value="es">España</option>

<option value="co">Colombia</option>

<option value="ar">Argentina</option>

</select><br>

<button>Enviar</button>

</form>
